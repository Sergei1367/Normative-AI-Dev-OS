# Normative AI Dev OS — Landing Page

Премиальная лендинг-страница для проекта **Normative AI Dev OS** — governance-first RAG платформы для контролируемой AI-разработки ПО.

## Технологии

- HTML5 (семантическая верстка)
- CSS3 (dark premium design, glassmorphism, градиенты, glow-эффекты)
- Vanilla JavaScript (scroll reveal, мобильное меню, копирование email)

## Структура

```
index.html
style.css
script.js
README.md
public/images/
  normative-ai-dev-os-architecture.png
```

## Раздел

1. **Header** — фиксированная навигация с glassmorphism
2. **Hero** — заголовок, теги, CTA-кнопки
3. **Problem** — 4 ключевые проблемы текущего подхода к AI-разработке
4. **Solution** — как платформа решает эти проблемы
5. **Architecture** — визуальная схема архитектуры + 8 pillars
6. **Governance Core** — 4 ключевых компонента governance
7. **Enterprise Readiness** — 6 enterprise-возможностей
8. **Tech Stack** — две плоскости: Control Plane и Compute Plane
9. **CTA** — призыв к действию с копированием email
10. **Footer** — ссылки и копирайт

## Использование

Откройте `index.html` в любом современном браузере.
